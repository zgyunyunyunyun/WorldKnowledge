## 🔐 修改默认密码（重要！）

部署前请务必修改默认密码为自己的强密码：

1. 用代码编辑器打开 `index.html`
2. 搜索 `const PASSWORD_HASH`
3. 将哈希值替换为你自己的密码哈希：

```javascript
// 1. 打开浏览器控制台（F12），粘贴运行以下代码生成哈希：
crypto.subtle.digest('SHA-256', new TextEncoder().encode('你的新密码'))
  .then(buf => Array.from(new Uint8Array(buf)).map(b => b.toString(16).padStart(2, '0')).join(''))
  .then(hash => console.log(hash))

// 2. 控制台会输出一串64位的十六进制哈希值，复制它

// 3. 在 index.html 中搜索 PASSWORD_HASH，替换为你的哈希：
const PASSWORD_HASH = '你生成的哈希值';
```

> ⚠️ **安全提示**：密码以 SHA-256 哈希形式存储，不会以明文出现在代码中。请选择一个自己容易记住但别人猜不到的强密码。

## 🧠 核心功能一览

### 基础知识管理
- **5大分类**：技术知识、产品/项目、阅读学习、想法灵感、思维模型
- **Markdown编辑**：支持实时预览，等宽字体编辑体验
- **标签系统**：自由打标，多维度检索
- **全文搜索**：按关键词搜索标题和正文
- **数据备份**：一键导出JSON，随时导入恢复

### 🤖 AI 思维工坊
- **深度追问**：对一条想法连续追问三层"为什么"
- **视角拓展**：从对立、外行、未来三个视角重新审视
- **关联发现**：自动发现知识库中的隐藏连接
- **行动转化**：将想法转化为具体可执行的计划

### 🧘 焦虑管理
- **想法倾倒**：把所有焦虑写下来，自动分类为"能做/以后做/放手"
- **认知重构**：识别认知偏差，用更平衡的视角重新看待
- **清空思绪**：5分钟引导式心灵清空练习

### ⌨️ 快捷键

## 🔗 如何连接真实AI

AI工坊默认使用**本地模拟模式**（内置思考模板），可以直接使用。

如需真实AI能力：

1. 打开AI工坊面板 → 底部"AI配置"区域
2. 填入你的API信息：
   - **API Key**：你的API密钥
   - **Endpoint**：兼容OpenAI格式的API地址（如 `https://api.openai.com/v1/chat/completions`）
   - **Model**：模型名称（如 `gpt-4o`、`deepseek-chat`）
3. 点击保存

> 配置仅存储在浏览器 `localStorage` 中，不会上传到任何服务器。

## 🎯 推荐使用流程

### 入门（第1周）
```
每天：
  ├─ 早上：打开工具，看看昨天的记录
  ├─ 随时：有想法就用 Ctrl+N 快速记录
  └─ 晚上：花5分钟整理今天的捕捉内容
```

### 进阶（第2-4周）
```
每周：
  ├─ 每天至少记录3条
  ├─ 周末：花20分钟做周回顾
  └─ 选2条最触动你的，用AI工坊做深度追问
```

### 深度使用（第2个月起）
```
每月：
  ├─ 用AI工坊做月度回顾
  ├─ 从知识库中选定一个主题，输出一篇完整文章
  └─ 重新审视分类体系，按需调整
```

## 💡 设计理念

1. **先捕捉，再整理**：不追求即时完美，先存下来再说
2. **碎片有价值，连接更有价值**：AI帮助发现跨领域的关联
3. **输出是最好的内化**：从碎片的"想法"到完整的"文章"
4. **焦虑是信号，不是敌人**：用结构化方法把焦虑转化为行动
5. **数据完全属于你**：所有数据存本地，可随时导出

---

## 🔒 隐私与安全

- 所有数据存储在浏览器 `localStorage`，不经过任何服务器
- 密码使用 SHA-256 哈希校验，不存储明文
- AI配置仅保存在本地
- 建议每1-2周导出一次JSON备份
- 不建议在其中存储高度敏感信息
